C Containers Generator

C Containers Generator is a quick and easy way to generate Containers of any data type in the C language. For user-defined data types some functions that operate on top of it may have to be provided such as comparing functions, copy functions and delete functions.

These Data Structures are not focused on performance, but rather on visualization.


How it works

Every container will be provided in two files. One is the source file container_name.c and another is the header file container_name.h.

You can choose to create a container of your own data type. If the type is a struct then it will be a pointer to that struct. This is because swapping values is done much easier then having an array of contiguous structs.


Features
  • It's all open sourced
  • Choose a name for your container
  • Choose a name for your header file and source file
  • Choose a prefix for your functions*
